个人博客搭建

可以借助github pages搭建服务，hexo搭建页面

设置github pages

  1. 登录github，新建仓库，注意仓库名字为username.github.io

  1. 构建和部署选择从分支发布（24年06.30后要用github actions 发布）

  1. Custom domain应该是可以设置域名，本次未处理

然后建好后测试仓库是否可用：https://lixiaoweil88.github.io/

用hexo搭建博客页面

安装

1.首先安装node.js,安装后注意配置为淘宝镜像

修改镜像：npm config set registry https://registry.npmmirror.com

查看镜像：npm config get registry

2.安装 hexo，和常用命令

安装：npm install hexo-cli -g

安装后查看版本 hexo -v

3.安装git

关联hexo和github 仓库

  1. 初始化文件夹为hexo项目，cd 目录然后执行 hexo init，初始化后可以执行hexo s 本地运行

  2. 初始化后修改_config.yml配置文件，关联到github

    deploy:
    type: ‘git’
    repository: https://github.com/lixiaoweil88/lixiaoweil88.github.io.git
    branch: master

  3. 部署到github,执行hexo g –d 会弹出登录窗口

    npm install hexo-deployer-git –save

    hexo g –d #推到github

写文章并发布到github

新建文章

hexo new “文章” 会在 博客目录source\_posts 下创建文章.md

hexo g 生成页面

hexo s 本地预览http://localhost:4000/

hexo g –d 推送到远程

插入图片

  1. 安装插件

    npm install hexo-asset-image –save

  2. 进入博客根目录找到node_modules-> hexo-asset-image-> index.js

    $(this).attr(‘src’, config.root + link + src);

    修改为

    $(this).attr(‘src’, data.permalink.split(‘example.com’)[1] + src);

  3. 在_config.yml配置文件将post_asset_folder:false改成true，hexo new “blog name”的时候会同时生成同名文件夹用来存放资源(将所有图片放在这个文件夹内)

  4. 配置typora 图片保存位置

  5. 本地预览和推送

    hexo clean #修改配置文件后执行

    hexo g

    hexo s

    hexo g –d
